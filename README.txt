================================================================
  LiveRecorder Pro — 抖音直播后台录制工具
================================================================

【项目概述】
LiveRecorder Pro 是一个抖音直播后台录制工具，支持：
  - 直连抖音直播间 FLV 流，自动下载保存
  - 按时间自动切割分段（每段独立文件）
  - FLV 自动转 MP4（需安装 ffmpeg）
  - 定时开始/定时停止录制
  - 多档画质选择（原画/超清/高清/标清/流畅）
  - 可调缓冲大小，应对网络抖动
  - 日间/黑夜主题切换


================================================================
【快速开始】
================================================================

方式一：原生桌面窗口（推荐，无需浏览器）
  1. 安装依赖：
     pip install -r requirements_desktop.txt
  2. 双击 start_desktop.bat
  3. 在弹出窗口中输入直播间 URL，点击录制按钮即可

方式二：浏览器控制面板（原版）
  1. 双击 start_panel.bat
  2. 浏览器访问 http://localhost:8766


================================================================
【安装步骤】
================================================================

必需：
  - Python 3.10 或以上
  - pip install pywebview requests

可选（推荐）：
  - pip install imageio_ffmpeg    （自动提供 ffmpeg，FLV→MP4 转码用）
  - 或从 https://ffmpeg.org 下载 ffmpeg 并添加到系统 PATH


================================================================
【Cookie 配置（重要！）】
================================================================

抖音直播间需要登录 Cookie 才能获取视频流。
请在 recorder_server.py 第 13 行修改 DOUYIN_SESSIONID：

  1. 用 Chrome 打开 https://live.douyin.com 并登录
  2. 按 F12 → Application → Cookies → live.douyin.com
  3. 找到 sessionid，复制其 Value
  4. 粘贴到 recorder_server.py 中的 DOUYIN_SESSIONID = "..."

注意：Cookie 会过期，如果提示"Cookie 已过期"，请重新获取。


================================================================
【控制面板说明】
================================================================

控制台标签页：
  - 直播间 URL：输入抖音直播链接（如 https://live.douyin.com/xxxxx）
  - 主播名：可选备注名，留空自动从 URL 提取
  - 画质：原画/超清/高清/标清/流畅（默认原画）
  - 缓冲大小：10/20/40/80 MB，网络不稳定时建议加大
  - 切割时长：0=不切割（单文件）；填数字则每 N 分钟切一个新文件
  - 定时开始：设置 X 分钟后自动开始录制（关闭浏览器也生效）
  - 定时停止：开始后自动在 X 分钟后停止

统计标签页：
  - 当前段时长、文件大小、分段数量、总录制时长、估算码率、磁盘空间

历史标签页：
  - 最近 50 条录制记录，可搜索/一键填充/删除

设置标签页：
  - 桌面通知、声音提示、主题切换、快捷键一览


================================================================
【输出文件】
================================================================

录制文件保存在 recordings/ 目录下（自动创建）。

文件命名格式：主播名_日期_时间.mp4
例如：张三_20260614_193000.mp4

如果录制过程中断或 ffmpeg 不可用，文件可能是 .flv 格式。
FLV 文件可用 VLC/PotPlayer 播放，也可用 ffmpeg 手动转换：
  ffmpeg -i 文件.flv -c copy 文件.mp4


================================================================
【键盘快捷键】
================================================================

  Ctrl + Enter   开始 / 停止录制
  Ctrl + S       手动切新段
  Ctrl + D       切换日间/黑夜主题
  Ctrl + 1~4     切换标签页
  Esc            关闭弹窗


================================================================
【桌面版特别说明】
================================================================

  - 桌面版将 Web 面板嵌入原生窗口，不需要打开浏览器
  - 关闭窗口会自动停止录制并保存当前分段
  - 如果录制中的最后一个分段未转码为 MP4，将保留 .flv 文件（可手动转换）
  - 连续录制多个主播时，旧的录制线程会在窗口关闭时全部清理


================================================================
【故障排查】
================================================================

Q: 提示 "Cookie 已过期"
A: 按上面【Cookie 配置】步骤重新获取 sessionid

Q: 提示 "ffmpeg NOT FOUND"
A: 安装 ffmpeg 并加入 PATH，或 pip install imageio_ffmpeg。
   没有 ffmpeg 不影响录制，只是文件保持 FLV 格式。

Q: 提示 "pywebview is not installed"
A: pip install pywebview

Q: 端口 8766 被占用
A: 之前的录制进程可能没退出，打开任务管理器结束 python.exe，
   或修改 recorder_server.py 和 desktop_app.py 中的端口号

Q: 窗口没有打开但没有报错
A: 确保 Windows 安装了 WebView2 运行时（Win11 自带，Win10 需手动安装：
   https://developer.microsoft.com/microsoft-edge/webview2/）

Q: 录制中断 / 画面不动
A: 可能是网络波动或主播下播，程序会自动尝试重连。如果持续中断，
   请检查 Cookie 是否有效，或尝试增大缓冲大小。
